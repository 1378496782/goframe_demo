# 飞书 CLI 使用指南

> 基于你的项目情况的个性化指南

## 🚀 飞书 CLI 能力总览

### 📋 核心功能列表

| 功能模块 | 能力 | 说明 |
|---------|------|------|
| 📧 **消息** | 发送/回复/转发 | 发送文本、富文本、消息卡片等 |
| 👥 **通讯录** | 查询用户/部门 | 搜索用户、查看组织架构 |
| 📅 **日程** | 查看/创建日程 | 管理日程、预订会议室 |
| 📝 **云文档** | 创建/编辑文档 | 文档、表格、多维表格 |
| ✅ **审批** | 审批处理 | 处理审批流程 |
| 🎯 **任务** | 任务管理 | 创建和管理任务 |
| 🎵 **妙记** | 会议纪要 | 查看会议录音和纪要 |

## 🎯 基于你的情况推荐路线

### 🌟 第一阶段：快速上手（推荐从这里开始）

**难度：⭐ 简单**

#### 1. 发送消息
**推荐理由：**
- 最简单，最快上手
- 可以先熟悉 CLI 流程
- 不需要太多配置

**应用场景：**
- 给自己发提醒
- 给测试群发通知
- 调试 API 时的消息通知

#### 2. 查看日程
**推荐理由：**
- 实用，每天都能用
- 了解日程管理能力

#### 3. 查看通讯录
**推荐理由：**
- 了解如何查询用户信息
- 为后续功能做准备

---

### 📈 第二阶段：结合你的项目

**难度：⭐⭐ 中等**

#### 1. 用飞书表格记录单词数据
**结合你的 Star 项目：**
- 把单词数据导出到飞书表格
- 方便查看和分享
- 可以用表格做简单统计

#### 2. 用多维表格做单词学习进度
**推荐功能：**
- 记录每个单词的学习进度
- 统计熟练度分布
- 可视化学习数据

#### 3. 用飞书提醒学习任务
**推荐功能：**
- 定时提醒学习
- 任务清单管理

---

### 🔧 第三阶段：高级集成

**难度：⭐⭐⭐ 较难**

#### 1. 集成到 GoFrame 项目
**想法：**
- 在你的 API 里集成飞书通知
- 单词更新时自动发提醒
- 用户注册时发欢迎消息

#### 2. 自动化工作流
**想法：**
- 定时从数据库导出单词到飞书表格
- 学习数据统计自动生成报告
